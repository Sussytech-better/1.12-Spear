# ⚔️ LungeFix

A lightweight Paper plugin that improves the **Spear Lunge enchantment** in Minecraft 1.21.11.

It allows players to activate Lunge by **left-clicking (even in the air)** instead of only triggering on hit, and adds a configurable cooldown + action bar UI.

---

## ✨ Features

- 🗡️ Uses real **Spear item**
- ⚡ Uses real **Lunge enchantment**
- 🪂 Activates on **air swing (left click)**
- ⏱️ 4-second configurable cooldown
- 📊 Live action bar cooldown display
- 🔊 Optional sound effects
- ⚔️ Scales with enchant level

---

## 🎮 How it works

Normally in Minecraft 1.21.10 and below:

- Lunge only triggers when you HIT an entity

With LungeFix:

- You can activate Lunge by **swinging in the air**
- Movement is applied instantly
- Cooldown prevents spam
